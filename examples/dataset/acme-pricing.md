# Acme Cloud API — Pricing

## Plans

### Basic — $29/month

- 500 requests per minute
- API key authentication only
- 2 regions (US-East, EU-West)
- Email support (48h response)
- API key validity: **12 months** (must be renewed before expiration)
- 5 GB storage included
- Standard SLA (99.9% uptime)

### Pro — $99/month

- 5,000 requests per minute
- API key + OAuth2 authentication
- 4 regions (US-East, US-West, EU-West, AP-Southeast)
- Priority support (4h response)
- Cross-region replication
- IP allowlisting
- API key validity: **24 months**
- 50 GB storage included
- Enhanced SLA (99.95% uptime)
- Webhook support

### Enterprise — Custom Pricing

- Custom rate limits
- Dedicated infrastructure
- Custom regions and data residency
- Dedicated support engineer (1h response)
- HIPAA compliance with BAA
- Unlimited API key validity
- Custom storage
- Premium SLA (99.99% uptime)
- SSO and SAML integration

## Billing

All plans are billed monthly. Annual billing is available at a 20% discount.

Overages beyond the included storage are billed at $0.10 per GB per month.

## Free Trial

New accounts receive a **14-day free trial** of the Pro plan. No credit card required. After the trial, accounts are downgraded to Basic unless a payment method is added.

## Usage Dashboard

Real-time usage metrics are available in the Dashboard:
- Requests per minute/hour/day
- Storage consumption
- Bandwidth usage
- Error rates by endpoint
